# Lean Bulk Meal Plan — PWA

Low-carb, high-protein weekly meal planner with macro tracking.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click "New Project" → Import this repo → Deploy
4. Open the live URL on your phone
5. Tap "Add to Home Screen" to install as an app

## Local Development

```bash
npm install
npm run dev
```

Built with React + Vite + vite-plugin-pwa.
